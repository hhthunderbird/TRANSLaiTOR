# Metrics MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an append-only JSONL metrics log that records one entry per `c.ps1` invocation, and a `cstats.ps1` CLI that summarises latency, mode distribution, cache hit rate, and compression ratio so we can quantify whether the local refiner actually saves tokens.

**Architecture:** Three pure-function helpers (`Add-MetricEntry`, `Read-MetricsFile`, `Get-MetricsSummary`) live in `cprompt.psm1` next to the existing JSONL-style `Add-HistoryEntry` helper — same module, no extra import in `c.ps1`. The CLI script `cstats.ps1` reads `~/.cprompt/metrics.jsonl`, computes the summary, and pretty-prints a table. `c.ps1` captures timestamps and char counts around the refiner and compiler stages and appends one metric entry just before exit. Failures in the metrics path are swallowed so they never break the user-facing run.

**Tech Stack:** PowerShell 5.1+, Pester (existing v3 style — `Should Be`, no dashes), `ConvertTo-Json -Compress` for serialization, `Measure-Object` for percentiles, `TestDrive:\` for filesystem isolation in tests.

**Branch:** `feat/metrics` off `main` (currently `8d969bf`).

The metrics log lives at `$env:USERPROFILE\.cprompt\metrics.jsonl` (Windows) — same parent directory as the existing history file, but a distinct filename so neither tool truncates the other's data. Tests inject a temp path via the `-Path` parameter.

**Entry schema (one JSON object per line):**

```json
{
  "ts": "2026-05-13T19:42:11.1234567Z",
  "model": "prompt-compiler",
  "refinerModel": "prompt-refiner",
  "mode": "questions",
  "inputChars": 213,
  "refinedChars": 487,
  "xmlChars": 1024,
  "refinerMs": 1280,
  "compilerMs": 3940,
  "totalMs": 5220,
  "cacheHit": false,
  "flags": { "Raw": false, "NoRefine": false, "Send": false }
}
```

`mode` ∈ `{ "raw", "passthrough", "questions", "skip", "cache" }`:
- `"raw"`: `-Raw` or `-NoRefine` was set — refiner was skipped on purpose.
- `"skip"`: refiner attempted but unavailable/failed, fell back silently to raw.
- `"passthrough"`: refiner ran and classified the input as sharp.
- `"questions"`: refiner ran and asked 1–3 clarifying questions.
- `"cache"`: compiler skipped because the cache key hit.

For `"cache"` entries `compilerMs` is `0` and `xmlChars` is the cached XML length.

function Add-MetricEntry {
    [CmdletBinding()]
    param(
        [Parameter(Mandatory)][string]$Path,
        [Parameter(Mandatory)][hashtable]$Entry
    )
    $dir = Split-Path -Parent $Path
    if ($dir -and -not (Test-Path -LiteralPath $dir)) {
        New-Item -ItemType Directory -Path $dir -Force | Out-Null
    }
    if (-not $Entry.ContainsKey('ts')) {
        $Entry['ts'] = (Get-Date).ToUniversalTime().ToString('o')
    }
    $line = ConvertTo-Json -InputObject $Entry -Compress -Depth 6
    Add-Content -LiteralPath $Path -Value $line -Encoding UTF8
}

function Read-MetricsFile {
    [CmdletBinding()]
    param([Parameter(Mandatory)][string]$Path)
    if (-not (Test-Path -LiteralPath $Path)) { return @() }
    $entries = @()
    foreach ($line in (Get-Content -LiteralPath $Path -Encoding UTF8)) {
        if ([string]::IsNullOrWhiteSpace($line)) { continue }
        try {
            $entries += ($line | ConvertFrom-Json)
        } catch {
            # Skip lines that fail to parse — partial writes, corruption, etc.
        }
    }
    return $entries
}

```powershell
function Get-MetricsSummary {
    [CmdletBinding()]
    param(
        [Parameter(Mandatory)][AllowEmptyCollection()][object[]]$Entries
    )

    $summary = [ordered]@{
        Count               = $Entries.Count
        CacheHitRate        = 0.0
        LatencyP50          = 0
        LatencyP95          = 0
        AvgCompressionRatio = 0.0
        ModeCounts          = @{}
    }

    if ($Entries.Count -eq 0) { return [pscustomobject]$summary }

    # Cache hit rate.
    $cacheHits = @($Entries | Where-Object { $_.mode -eq 'cache' }).Count
    $summary.CacheHitRate = [math]::Round($cacheHits / $Entries.Count, 4)

    # Mode counts.
    $modeCounts = @{}
    foreach ($e in $Entries) {
        $m = [string]$e.mode
        if (-not $m) { $m = 'unknown' }
        if (-not $modeCounts.ContainsKey($m)) { $modeCounts[$m] = 0 }
        $modeCounts[$m] = $modeCounts[$m] + 1
    }
    $summary.ModeCounts = $modeCounts

    # Latency percentiles over totalMs.
    $latencies = @($Entries |
        Where-Object { $_.totalMs -ne $null } |
        ForEach-Object { [int]$_.totalMs } |
        Sort-Object)
    if ($latencies.Count -gt 0) {
        $summary.LatencyP50 = $latencies[[math]::Max(0, [math]::Ceiling(0.50 * $latencies.Count) - 1)]
        $summary.LatencyP95 = $latencies[[math]::Max(0, [math]::Ceiling(0.95 * $latencies.Count) - 1)]
    }

    # Average compression ratio (xmlChars / inputChars) over entries with both > 0.
    $ratios = @($Entries |
        Where-Object { $_.inputChars -ne $null -and [int]$_.inputChars -gt 0 -and $_.xmlChars -ne $null } |
        ForEach-Object { [double]$_.xmlChars / [double]$_.inputChars })
    if ($ratios.Count -gt 0) {
        $sum = 0.0
        foreach ($r in $ratios) { $sum += $r }
        $summary.AvgCompressionRatio = [math]::Round($sum / $ratios.Count, 4)
    }

    return [pscustomobject]$summary
}
```

param(
    [int]$Last = 0,
    [string]$Path = (Join-Path $env:USERPROFILE '.cprompt/metrics.jsonl')
)

$ErrorActionPreference = 'Stop'
Set-StrictMode -Version Latest

$here   = Split-Path -Parent $MyInvocation.MyCommand.Path
$module = Join-Path $here 'cprompt.psm1'
Remove-Module cprompt -ErrorAction SilentlyContinue
Import-Module $module -Force

$entries = @(Read-MetricsFile -Path $Path)
if ($Last -gt 0 -and $entries.Count -gt $Last) {
    $entries = $entries[($entries.Count - $Last)..($entries.Count - 1)]
}

$summary = Get-MetricsSummary -Entries $entries

Write-Host "Metrics file : $Path"
Write-Host "Entries      : $($summary.Count)"
if ($summary.Count -eq 0) { return }

Write-Host ("Cache hits   : {0:P1}" -f $summary.CacheHitRate)
Write-Host ("p50 totalMs  : {0}" -f $summary.LatencyP50)
Write-Host ("p95 totalMs  : {0}" -f $summary.LatencyP95)
Write-Host ("Avg xml/input: {0:N2}" -f $summary.AvgCompressionRatio)
Write-Host ''
Write-Host 'Mode counts:'
foreach ($mode in ($summary.ModeCounts.Keys | Sort-Object)) {
    Write-Host ("  {0,-12} {1}" -f $mode, $summary.ModeCounts[$mode])
}
